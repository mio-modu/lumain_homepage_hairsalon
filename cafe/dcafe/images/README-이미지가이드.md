- 이미지가 없어도 각 자리에 브랜드 톤의 그라데이션 배경이 깔려 레이아웃이 깨지지 않습니다.
- `visit-map.jpg`는 실제 지도 캡처 이미지를 넣으면 좋고, 클릭하면
  `https://map.kakao.com/?q=대전 유성구 상대동 477-7 그린브라우니` 로 새 탭 연결됩니다.
