`sig-einspanner.jpg` 등으로 교체 가능합니다(요청 주세요).

## ⏳ 추가 예정 — 약도(略圖) 이미지
- 오시는 길 섹션에 넣을 **약도 이미지**를 주시면 반영 예정.
- 파일명 예: `visit-map.jpg` (약도) — 지도자리에 배치하고 카카오맵 링크 연결.
- 약도를 주시면 현재 야간 외관 사진은 갤러리/다른 자리로 옮기거나 병행 배치 가능.
