# 이미지 채우는 방법

이 폴더에 아래 파일명으로 이미지를 넣으면 `index.html`이 자동으로 인식해서 보여줍니다 (파일이 없으면 로즈핑크 placeholder 박스가 대신 표시됩니다). 코드를 수정할 필요 없이 파일만 넣으면 됩니다.

Midjourney, DALL-E, Stable Diffusion 등 AI 이미지 생성 도구에 아래 프롬프트를 그대로 입력해서 생성한 뒤, 결과 파일을 지정된 이름으로 저장하세요.

## 진행 상황

| 파일명 | 용도 | 상태 |
|---|---|---|
| `hero.png` | Hero 배경 | ✅ 완료 |
| `about.png` | About 섹션 | ✅ 완료 |
| `designer-1.png` | 디자이너 1 | ✅ 완료 |
| `designer-2.jpg` | 디자이너 2 | ⬜ 미완성 |
| `designer-3.jpg` | 디자이너 3 | ⬜ 미완성 |
| `gallery-1.png` | 갤러리 (발레아쥬 웨이브) | ✅ 완료 |
| `gallery-2.png` | 갤러리 (슬릭 단발컷) | ✅ 완료 |
| `gallery-3.jpg` ~ `gallery-6.jpg` | 갤러리 나머지 4장 | ⬜ 미완성 |
| `map.jpg` | 오시는 길 지도 | ⬜ 미완성 (AI 생성 아님, 카카오맵/네이버맵 캡처 권장) |

`Generated Image July 13, 2026 - 5_06PM (4).png`는 소프트 컬 사진인데, 생성 결과 이미지 안에 프롬프트 텍스트가 깨진 글자로 함께 박혀 나와서 사용하지 않고 폴더에 남겨뒀습니다. 다시 생성해서 `gallery-3.jpg`로 저장해주시면 반영해드릴게요.
